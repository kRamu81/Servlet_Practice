## Web Development Using Servlets & JSP:

# Create Web Project:
1. New file-> Dynamic Web Project
2. Project Name: StudentWebApp
3. Target Runtime->Apache Tomcat
4. Finish

# Configure Tomcat in Eclipse:
1. Window->Show View->Servers
2. Click No servers available
3. Select Apache Tomcat v9
4. Browse Tomcat folder
5.Finish

# First Servlet Example:
Create Servlet:
1. Right click project
2. New->Servlet
3. Package: com.student
4. ServletName: HelloServlet
5.Finsih
